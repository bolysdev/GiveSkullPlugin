# Русский: GiveSkullPlugin
Легковесный и эффективный плагин для Minecraft-серверов, решающий проблему получения голов игроков.

🌟 Почему это важно?
Начиная с последних обновлений Minecraft, многие популярные плагины (включая EssentialsX) часто сталкиваются с багом, при котором вместо головы игрока выдается стандартная голова Стива или Алекса. GiveSkullPlugin обходит эту проблему, напрямую обращаясь к актуальному API для корректной подгрузки голов.

🚀 Особенности
Поддержка версий: от 1.21 до актуальной 1.21.x (26.10).

Мультиязычность: Плагин поддерживает Русский, English, Қазақша и Deutsch.

Простота: Ничего лишнего, только работающий функционал.

🛠 Команды
/skull — Получить собственную голову.

/skull <ник> — Получить голову указанного игрока.

/skull lang <language> — Смена языка интерфейса.

🔑 Права (Permissions)
skull.give — Доступ к основной команде выдачи голов (по умолчанию у OP).

skull.language — Возможность менять язык плагина (по умолчанию у OP).

# English: GiveSkullPlugin
GiveSkullPlugin is a lightweight plugin designed to fix the common issue where head items show default skull (Steve/Alex) instead of the actual player's skull.

🌟 Features
Versions: Supports 1.21 up to 1.21.x.

Multi-language: Supports English, Russian, Kazakh, and German.

Reliable: Uses an alternative method to fetch textures, bypassing EssentialsX skin bugs.

🛠 Commands
/skull — Get your own head.

/skull <nickname> — Get the head of the specified player.

/skull lang <language> — Change the interface language.

🔑 Permissions
skull.give — Access to the main head-give command (default for OP).

skull.language — Ability to change the plugin language (default for OP).
